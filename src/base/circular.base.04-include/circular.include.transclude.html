cc-transclude, when used within an include, includes the content that was originally inside the node before the external file was inserted. This basicly allows you to hardcode content in a node, and apply a external file to format it later. Notice how this page does not use a cc-context or a javascript Model at all - all the data is hardcoded inside the html, and formatted using cc-transclude within the external file :-)
