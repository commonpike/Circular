cc-loop has some additional tags and classes to manage row handling. Elements with attributes cc-first and cc-last will only be display for the first and the last element in the loop. Elements with tag cc-loop-index="3" will only be displayed on the 3rd index of the loop. Elements with tags cc-loop-odd and cc-loop-even will only be displayed in odd and even entries of the loop.

Similarly, cc-loop adds some css classes to handle row styling: .cc-loop-first,.cc-loop-last,.cc-loop-odd,.cc-loop-even.
