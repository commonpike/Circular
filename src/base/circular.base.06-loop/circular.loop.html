cc-loop requires to have an {{expression}} as a value that resolves to an object or array. It repeat the children of the node for each element in the object or array, and set the context of each generated child to, by default, the value of that element.

So <div cc-loop="{{[1,2,3]}}">{{#this}}</div> will simply print 123 for you, and <div cc-loop="{{{foo:1,bar:2,quz:3}}}">{{#this}}</div> will print the same.
