By default, cc-loop puts whatever you are looping in the context of each loop item. Within each item, {{#this}} would refer to the looped item, and {{#foo}} would refer to the property 'foo' of that item, if it's an object.

That can quickly become confusing, especially within nested items. By using cc-loop-as, you can specify a different name for the word this. For example <div cc-loop="#regions"> <div cc-loop="#this"> <ul cc-loop="#this"> <li>{{#this}}</li> </ul> </div> </div> Could be written as <div cc-loop="#regions" cc-loop-as="region"> <div cc-loop="#region" cc-loop-as="country"> <ul cc-loop="#country" cc-loop-as="city"> <li>{{#city}}</li> </ul> </div> </div> When using cc-loop-as, {{#this}} becomes quite useless.
