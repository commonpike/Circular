By default, cc-input elements both read the targets value into the input element and write changes in the input element back to the target.

Using cc-input-dir, you can set it to only read or write.


read only:
write only:
values for reading:
{{#mystring}}
{{#mynumber}}
{{#myboolean}}
values for writing:
{{#mystring}}
{{#mynumber}}
{{#myboolean}}
