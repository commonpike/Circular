You can change the default 'cc-' prefix of all custom attributes to something else (like the html5 compliant 'data-cc-'). Below example uses 
Circular.init({
	modules : { attrprefix	: 'square-' }
})
If you do this, all attribute names in your html that originally started with cc- need to start with square- instead.
#message='hello' #message='goodbye'
