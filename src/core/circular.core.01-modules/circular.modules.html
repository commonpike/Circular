Everything in Circular is a module. Even the modules module is a module. Core and Base modules are included in circular.js. Contrib modules need to be loaded by yourself.

You can also write your own modules, simply by writing new CircularModule('foo',{ config : { bar : [] } } Modules can, amongst others, define custom attributes and comments.
