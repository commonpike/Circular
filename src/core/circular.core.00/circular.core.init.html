After loading the Circular javascript, you usually call Circular.init() with a 'config object' that sets various options for all the modules, at once. This page uses Circular.init({ log : { debug : true }, engine : { debug : true } }); which sets two options, one for the 'log' module and one for the 'engine' module. Init happens on document.ready. If you don't use Circular.init(), init will happen automatically with default options on document.ready.
