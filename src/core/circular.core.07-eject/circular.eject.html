The eject module is a built in mech to move nodes out of the visible document tree. Other modules, like @if, use this. You can call 'eject' from your own module using Circular.eject.flag() and Circular.eject.unflag().

When ejected, the node is moved to #cc-ejected, and is replaced by a <!--cc-eject comment which makes sure the original content is still parsed as Circular cycles through the document tree - anytime something changes. So the attribute that 'flagged' the node for ejection can 'unflag' it again when appropriate.

The module has one attribute, cc-eject, which is basicaly the same as cc-if (but without the cc-else).
