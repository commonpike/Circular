You can use {{expressions}} in text content. They are usually replaced by <span> elements with a cc-content attribute, so they can be watched and updated when things change.

If an expression is the only thing nested inside an element, no <span> element is created, but the cc-content attribute is set on that parent element itself.
If however there is whitespace surrounding the expression, a <span> element would be created to honor the whitespace. To have Circular eat whitespace around inline expressions, set Circular.engine.greedy to true, like this: Circular.init({ engine : { greedy : true } });
