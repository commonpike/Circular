The attribute cc-content copies the value of the attribute into the node, overwriting anything inside it.

This is mainly useful if you have {{expressions}} in the attribute. Though you can use expressions in text content, they will shortly show up before the document is parsed by Circular. Also, expressions inside text content are replaced by <span> elements with a cc-content attribute anyway .. so .. might as well use cc-content yourself.
