If you use an expression inside a cc-context attribute, and that expression evaluates to a string, that string is used as a context. This way, you can create references within your data.

The below snippet uses this as a model: var Data = { page : { message : 'Welcome aboard', user : 'Data.user.current' }, user : { current : { title : 'passenger' } } }

{{#message}}, {{#title}}
