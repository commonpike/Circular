While cycling your document, the @engine keeps a running 'context' using the @context module. Every expression in your document is parsed using that context. For example, if the context is the string "Beatles.songs", {{#links}} really means "Beatles.songs.links" (and {{#this}} just means "Beatles.songs").

The context module provides an attribute, cc-context, which you use to define the context on a node and all its children. The below snippet uses this object model: var Model = { foo : 'This is Model.foo', bar : { quz : 'This is Model.bar.quz' } } Notice how the context is 'just a string'. You do not need to use an {{expression}} to set the context, unless you want it to be parsed.
